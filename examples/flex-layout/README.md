# flex-layout

## What it shows

Box layout via Yoga: `flexDirection`, `justifyContent="space-between"`, `alignItems="center"`, `gap`, and uniform `padding`. Four small panels each exercise one knob so you can see them in isolation.

## How to run

```sh
pnpm --filter @vue-ink-examples/flex-layout start
```

## What to look at first

- [`flex-layout.vue`](./flex-layout.vue) — each `<Box>` block is annotated with the prop it demonstrates.
