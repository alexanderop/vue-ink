# borders

## What it shows

All five border styles (`single`, `double`, `round`, `bold`, `classic`), `borderColor`, per-side toggles (`borderLeft={false}`, `borderRight={false}`), per-side colors (`borderTopColor`, `borderRightColor`, `borderBottomColor`, `borderLeftColor`), and `borderDimColor`.

## How to run

```sh
pnpm --filter @vue-ink-examples/borders start
```

## What to look at first

- [`borders.vue`](./borders.vue) — each `<Box>` block exercises one border feature.
