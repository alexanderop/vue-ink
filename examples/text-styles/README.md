# text-styles

## What it shows

The six text-style flags on `<Text>`: `bold`, `italic`, `underline`, `strikethrough`, `inverse`, `dimColor`. The last row stacks several styles on one node to show they compose.

## How to run

```sh
pnpm --filter @vue-ink-examples/text-styles start
```

## What to look at first

- [`text-styles.vue`](./text-styles.vue) — one `<Text>` per style, plus the combined example at the bottom.
