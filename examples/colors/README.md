# colors

## What it shows

Every named foreground and background color rendered with `<Text color="…">` and `<Text backgroundColor="…">`. Useful as a visual smoke test for ANSI color output on your terminal.

## How to run

```sh
pnpm --filter @vue-ink-examples/colors start
```

## What to look at first

- [`colors.vue`](./colors.vue) — `v-for` over the color list, applying `color` then `backgroundColor`.

## Related docs

- [`<Text>` component](../../packages/docs/api/components.md#text) — `color`, `backgroundColor`, color names and hex.
