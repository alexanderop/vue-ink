# @vue-ink/components

First-party Vue components for [vue-ink](https://github.com/alexanderop/vue-ink): `<Box>`, `<Text>`, `<Newline>`, `<Spacer>`, `<Static>`, `<Transform>`.

Most users should install `vueink`, which re-exports these alongside the renderer.
