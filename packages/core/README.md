# @vue-ink/core

Terminal DOM, layout (Yoga), and ANSI output pipeline that powers [vue-ink](https://github.com/alexanderop/vue-ink). **No Vue dependency** — this is the framework-agnostic rendering core.

Most users should install `vueink` instead. Reach for this package directly only if you're building tooling that needs the layout/output primitives without Vue.
