# @vue-ink/renderer

Vue custom renderer for [vue-ink](https://github.com/alexanderop/vue-ink). Bridges Vue reactivity to the terminal DOM in `@vue-ink/core`. Ships `render()`, `renderToString()`, `measureElement()`, and every composable (`useInput`, `useFocus`, `useApp`, …).

Most users should install `vueink`, which re-exports everything here plus the components in `@vue-ink/components`.
